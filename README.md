# DevForge Bash Commands

Welcome to your custom DevForge Terminal. Here are the commands you can use:

## Platform Commands
- `login`          : Authenticate with the DevForge platform.
- `whoami`         : Show the currently logged-in user.
- `rep`            : View your reputation points.
- `tickets list`   : List all open tickets.
- `tickets view ID`: View details of a specific ticket.
- `tickets create` : Create a new ticket.
- `projects`       : List all your projects.
- `snippets`       : List your code snippets.
- `config set-url` : Set a custom base URL (useful for ngrok/production).

## System Commands
- `ls`             : List files in current directory.
- `cd [dir]`       : Change directory.
- `pwd`            : Print working directory.
- `clear`          : Clear the terminal screen.
- `date`           : Show current date/time.
- `echo [text]`    : Print text to terminal.

## Features
- **Premium Purple Theme**: Matches the DevForge web experience.
- **Git Bash Layout**: Familiar structure (`bin/`, `etc/`, etc.).
- **Standalone Emulator**: No need to use PowerShell or CMD.
